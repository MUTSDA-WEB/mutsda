# MUTSDA Church Website

This is the official repository for the MUTSDA (MUrang'a University of Technology Seventh-day Adventist) Church website.

## Project Overview

This project is a web application designed to serve the MUTSDA church community. It will provide information about church events, sermons, ministries, and other relevant resources.

## Tech Stack

- **Frontend:** React, Vite
- **Styling:** TBD (e.g., Tailwind CSS, Material-UI)
- **Backend:** TBD (e.g., Firebase, Node.js/Express)

## Getting Started

### Prerequisites

- Bun

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd mutsda
   ```
3. Install dependencies:
   ```bash
   bun add .
   ```

### Development

To start the development server, run:

```bash
bun dev 
```

This will start the Vite development server with Hot Module Replacement (HMR) enabled.
